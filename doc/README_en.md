## Quick Start

### For Synchronizable OJs (Except NowCoder)
1. Works instantly if you have installed the script and linked these accounts on VJudge .

2. Navigate to the VJudge website; a control panel will appear on the side.

3. Select the OJs you wish to sync and click "One-click Sync" (ä¸€é”®åŒæ­¥).

4. To stop the synchronization, simply refresh the page (the "stop" button hasn't been implemented yet!).
![](panel.png) 
### For Experimental OJs (Currently NowCoder only)
Syncing is not recommended for now due to several ongoing issues. (Issues include rate limiting and missing problem data; these will be addressed in future versions.)
