# VJudge の AC自动机

本脚本用于 **一键同步 VJudge 上已绑定的其他 OJ 数据**，  
方便统一统计个人的过题情况与做题进度。

---

## 当前支持的 OJ

-[x] 洛谷
-[x] Codeforces
-[x] AtCoder
-[x] QOJ
- [ ] 牛客
