**前言：**
由于VJ平台功能十分强大，拥有团队/题单/过题详情/AC代码导出等众多实用功能，
故想做个工具将其他OJ的过题数据同步到VJ平台，便于数据的统一管理

# VJudge の AC自动机

本工具便是用于 **一键同步 VJudge 上已绑定的其他 OJ 数据**，  
目前支持市面上大部分主流oj

## 快速使用

### 过题一键同步
**在VJudge上绑定过账号就可以实现一键同步**

1. 安装好脚本/扩展,进入VJ,旁边就会出现一个面板/弹窗
2. 勾选要同步的OJ,点击"一键同步"即可
3. 要停止同步,刷新页面/关闭窗口即可(终止功能懒得写了)
- 注：首次同步可能因为题数较多而耗时较长，请耐心等待哟~，速率问题暂未解决
